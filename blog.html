Important FAQ?

What is the Document Object Model (DOM)?

Document Object Model or DOM is the Object structure of the browser through which JavaScript can interact with the browser UI . Thus it helps us to maintain a good interactivity of the webpage.

How do you Select an Element From the DOM?

To select an element from the dom there are several ways to do that. To select a tag we use the DOM method getElementByTagName() . We can select a specific element that has an id using getElementById() DOM method. To select a specific element of a class name we use the getElementByClassName(). A single method is widely used which is querySelector() helps us to select a specific element of any type whether it can be class or id.

What is Event Delegation in the Context of the DOM, and Why it is useful?

When we give a specific event handler into an element and the parent node of that element also has another event handler then if the child element is clicked, both of the event runs. this is called event delegation. To stop this we use stopPropagation() function.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element't attributes or styles we add an eventListener to that specific element. inside the eventListener we explicitly mention what to change and what to edit. we use the setAttribute() to change attributes and we use the classList.add() to add a class and classList.remove() to remove a class from an element.
